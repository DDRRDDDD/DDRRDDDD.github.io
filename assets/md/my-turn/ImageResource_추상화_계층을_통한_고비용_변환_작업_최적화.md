#### ❝동일한 이미지를 여러 형태로 가공해 보여줄 때 발생하는 중복 연산을 제거하여 CPU 점유율을 낮췄습니다.❞

#### 🚨 문제(Problem)
> * 프로필 이미지 등 반복 사용되는 자원에 대해 Clipping, Filtering 등 고비용 렌더링 연산이 위젯 트리 곳곳에서 중복 실행됨.

#### 🛠 해결(Solution)
> * **추상화 계층(Layer) 도입** : 원본 이미지 로드와 변환 로직을 캡슐화한 `ImageResource` 클래스 설계.
> * **싱글톤 캐싱** : 변환된 이미지 객체(ui.Image)를 메모리에 캐싱하여, 동일한 요청 시 연산 없이 즉시 반환(O(1))하도록 구조화.

코드 필요**예시 코드: ImageResource 클래스**

#### 📈 성과(Outcome)
> * 이미지 변환 작업을 세션당 **1회로 제한**.
> * 불필요한 중간 객체 생성을 막아 메모리 효율성 개선 및 GC 부하 감소.
